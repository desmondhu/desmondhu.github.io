新建博文

进入personal web>hexo>文件夹下面，鼠标点击gitbashhere，执行： 

hexo n "博客文章名字"

之后在source文件下打开刚刚新建的.dx文件进行编辑。

Markdown编辑

有以下几个注意事项

  1. 按照markdown语法对md文件进行编辑。首先添加标签。与技术相关主要使用英文，标准形式如：LabVIEW、Matlab、Hexo、Markdown、点滴…
  2. 对于初始生成的模板，每次结尾不要有空格，直接键入enter键换行。这里的格式错误也会一切发布hexo g生成错误。
  3. 在文件中添加代码行时，需要中间隔开一行，因为代码行比较宽，之后按下Tab键。代码在markdown左侧编辑环境下颜色为绿色。
  4. 注意正文换行的时候需要两个空格键。

发布到Hexo

hexo g                                   #生成网页
hexo s                                   #本地预览  
hexo d                                   #发布

一键部署 

hexo d -g

题外话（为什么开始写）

  1. 互联网科技发展很快，而这个涉及到一些相关东西，markdown标记语言、git一个开源的分布式版本控制系统、github个人免费远程仓库。通过搭建博客的过程中对他们有了稍微了解，虽然只停留在使用一些简单的功能按时有一个特点却是出乎意外。通过github：可以接触到一流的软件项目，甚至为一个项目作出提议，让你和许多伟大的程序员同步，我也发现许多开源网站或者个人将自己的项目托管在github上面，这真是令人兴奋！
  2. 促进自己将自己的技术项目整理。发布到网上不是目的，也许是整理下思路，或者向别人介绍我做过的项目更加重要。
  3. 比微博，空间，一些论坛的博客有更大的自由度和控制权。
  4. 方便自己记录生活点滴
  5. github还没被墙

致谢

首先要感谢这个开源时代，许多人分享他们的项目，技术；其次感谢Hexo！一个简单、快速、强大的Node.is静态博客框架，感谢Next提供了简洁的next主题。开始我的博客之旅~
